### Compatible OS 
- Linux 
- Mac OS

### Description 
[[中文版](./README_zh.md)] Crawer for downloading pictures from [mzitu](https://www.mzitu.com/).

### Dependency 
    # scrapy and python3.
	$ sudo pip3 install scrapy

### Save folder 
   Modify the save path in "mzitu/settings.py" line 12.

### Usage 
	Under the mzitu/ directory
	$ scrapy crawl mzitu
