# 描述 #
<li><a href="README.md">English</a></li>
从[妹子图](https://www.mzitu.com/)网站下载所有妹子图片。

# 依赖 #
    # scrapy 
	$ sudo pip3 install scrapy

# 图片保存位置 #
   到mzitu/settings.py中修改第13行处图片存储路径值

# 使用 #
	退回到项目主目录
	$ scrapy crawl mzitu

注意:请使用python3，python2已于2020.01.01正式退役。
